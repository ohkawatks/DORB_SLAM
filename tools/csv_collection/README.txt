Summary

This tools collect performance, which was print from measurment controller.

Usage : 

If would you like to see in div,please set following parameter.
Tools will print result to measurment raw data path.
 *collection_div [measurment raw data path] [measrurment raw data file]

If would you like to see in normal,please set following parameter.
Tools will print result to measurment raw data path.
 *collection_normal [measurment raw data path] [measrurment raw data file]

Build :

 Please do following STEP in div or normal folder.
 STEP1. mkdir build 
 STEP2. cmake ..
 STEP3. make
